A map script that uses elevation data to overlay TopoJson layers along particular contours. This map is using elevation contours from shapefile data from the USGS National Map Viewer. These elevation contours are used to draw shapes on top of particular contours.

Each shape encodes elevation information, which it displays when the mouse is hovered over the layer.

The map tileset is also from the USGS National Map Viewer.
